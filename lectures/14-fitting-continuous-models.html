Sidenote

Hats like \(\hat{\ }\) or \(\dot{\ }\) or \(\tilde{\ }\) are often used to denote an estimate of an expression.

Be careful: The specific notation can change from document to document.

One step beyond the ECDF and histogram

Recall that the definition for the ECDF is

\[\text{ecdf}(t, \boldsymbol{X}_N) = \sum_{i = 1}^N \frac{1}{N}I(X_i \leq t)\]

How would the ECDF change if we used

\[\text{pnorm}(t, X_i, .1) \text{ instead of } I(X_i \leq t)?\]

\[\text{ecdf*}(t, \boldsymbol{X}_N) = \sum_{i = 1}^N \frac{1}{N}\text{pnorm}(t, X_i, .1)\]

One step beyond the ECDF and histogram

ecdfstar <- function(t, data){
  outer(t, data, function(a,b){ pnorm(a, b, .1)}) %>% rowMeans
}
plot(ecdf(data), main = "")
curve(ecdfstar(x, data), add = TRUE, lwd = 3, col = "blue")

ecdfstar <- function(t, data, smooth){
